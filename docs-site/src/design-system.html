ℹ️
WASM-safe crates

The core crate set — marque-ism, marque-core, marque-capco — compiles to WASM without modification. Format extraction lives in marque-extract only.

⚠️
Citation integrity

Every rule citation must reference a real passage in the authoritative source. Fabricated or hallucinated citations are a correctness defect, not a style issue.

🚫
Do not call __engine_promote directly

The AppliedFix::__engine_promote constructor is engine-only. Calling it from rule crates bypasses the confidence-threshold gate and corrupts the audit log.

📎
Performance targets

Interactive use: p95 ≤ 16ms on typical inputs (SC-001). Batch: linear throughput scaling (SC-005). All claims backed by committed Criterion benchmarks.
