&cp_object; v2021-NOV Package Description

&cp_businessObjectClass; (&cp_object;) consists of many files all packaged here for you. This is a simple guide to the layout of this package. Before attempting to follow the links in this guide, please ensure that you have extracted the files from the zip file; otherwise, the links will not work.

As this is the standalone package, none of the dependent specifications will be included and you will have to assemble the full package with an acceptable version of the dependent specifications that you wish to use.

  • CONTENTS

    The following components may be included in the package:

    • XYZ*_Rules.pdf

      A single human friendly document representation of all the Schematron rules for the XYZ Specification

    • XYZ*_Schema.pdf

      A single human friendly document representation of the schema for the XYZ Specification

    • XYZ*_XSL.pdf

      A single human friendly document representation of all the XSLs for the XYZ Specification

    • XYZ*_XSPEC.pdf

      A single human friendly document representation of all the XSPECs for the XYZ Specification

    • XYZ*CVEnums.pdf

      A single human friendly document representation of all the Common Value Enumeration Tokens and their Descriptions for the XYZ Specification

    • XYZ*Examples.pdf

      A single human friendly document representation of all the included examples for the XYZ Specification

    • Des*Xml.pdf

      The primary description and implementation guide for the XYZ Data Encoding Specification

    • Ces*Xml.pdf 

      The primary description and implementation guide for the XYZ  CVE Encoding Specification

    • Annex / FOUO Annex

      Additional information and guidance for the XYZ Specification that is either UNCLASSIFIED, but not publicly releasable or at the U//FOUO level

    • SECRET Annex

      Additional information and guidance for the XYZ Specification, at the SECRET level (either NOFORN or RELTO)

    • TetragraphTaxonomy.pdf

      A single human friendly document representation of the Tetragraph Taxonomy XML file

  • Documentation

    This includes the document for this specification and the applicable signature memo associated with it.

    • ISM-Rollup version 2021-NOV
      • DesISMRollupXml.pdf
      • SignatureMemo.pdf

  • Examples

    This specification does not include a schema file and does not include example instance documents. If dependent specifications have example files, they will not be included in this package. The dependent specification example files may be obtained from the respective specification package.

  • Schema

    This specification does not include a schema file. If dependent specifications have schema files, they will not be included in this package. The dependent specification schema files may be obtained from the respective specification package.

  • Schematron

    This specification does not include a schema file and does not include schematron files. If dependent specifications have schematron files, they will not be included in this package. The dependent specification schematron files may be obtained from the respective specification package.

  • XSL

    The XSL file to implement this specification excluding dependencies.

    • ISM-Rollup

      • ISM-Rollup XSL Files
      • ISM-Rollup_XSL.pdf

  • XSPEC

    The XSPEC file to implement this specification excluding dependencies.

    • ISM-Rollup

      • ISM-Rollup XSPEC Files
      • ISM-Rollup_XSPEC.pdf

  • manifest.md5

    This file contains an MD5 checksum of every file in the package. It can be used to determine if any of the files were corrupted in unzipping or if you have modified a file.

  • ISM-Rolluppackage.properties

    This file contains information concerning the version, classification, and build time for the package.
